## Priority Issues

1. [P1] Library + sidebar invisible to keyboard/assistive tech. .play is a div, rows/sb-items unfocusable, #status has no aria-live. Fix: button-ize .play with aria-label, roving tabindex on rows (Enter=play), aria-live="polite" on #status.
2. [P1] Armed-row trap corrupts combine selections. Plain click arms a row with the same .selected ring; joins next ⌘-click selection silently. Fix: distinct quieter treatment or drop armed concept; show selection bar whenever anything is ringed.
3. [P2] Low-contrast small text cluster. #98989f at 9.5–11.5px ≈2.6–2.9:1 (needs 4.5:1): stat labels, kbd hints, paths, sizes, np path. Fix: bump tertiary text to ≈#6e6e73 at small sizes or enlarge.
4. [P2] Esc inconsistent: doesn't close #detail or exit selection mode. Fix: add both to keydown chain.
5. [P2] Best feature undiscoverable: no visible ⌘F hint, main search placeholder omits tokens. Fix: ⌘F kbd chip in search field, token examples in placeholder.
6. [P3] Vintage-theme leaks: cream sidebar tint, #4a4233 version titles, warm rgba(60,45,25) shadows, dead .seg/.toggle CSS, header gradient literals. Fix: sweep remaining warm literals onto tokens.
7. [P3] Ignore terminology drift: "Remove from list" / "Hide a folder" / "Ignored folders" → standardize on "Ignore".

## Persona Red Flags

Alex (power user): can't arrow through library without starting playback; no keyboard rename/star/details; no shortcut reference anywhere; armed+⌘J is hidden modality.
Sam (screen reader/keyboard): cannot play any song; Tab skips list+sidebar; star/reveal buttons lack song context in accessible name; playing state is color-only; #98989f small text below AA.
Producer mid-session: elite happy path (⌘F→type→Enter→drag to DAW) but undiscoverable; failed play is silent dead air; newest-bounce-as-head can surface "v1 rough" over "v3 FINAL" with the pin fix hidden in the collapsed stack.

## Questions to Consider

1. What if ↑/↓ moved a Finder-style cursor (Space=audition, Enter=details) so the library, not just the transport, is keyboard-first?
2. Does the armed-anchor concept deserve to exist, or is it a combine-flow convenience leaking state into every click?
3. At 5,000 songs, does "Uncategorized" become the largest bucket — and what would make WavCave proud of the mess instead of labeling it?
